## 🎯 우선순위별 구현 계획

### Phase 1: 핵심 가치 기능 (즉시 구현)

#### 1.1 관리자 페이지 완성 ⭐⭐⭐
**목적**: 플랫폼 관리 및 운영 효율화
- `/admin/users` - API 연결 및 실시간 사용자 관리
- `/admin/posts` - 게시물 관리 UI 완성
- `/admin/reports` - 신고 처리 워크플로우
- `/admin/meetings` - 모임 관리 및 통계
- `/admin/gyms` - 체육관 승인 프로세스

#### 1.2 ELO 기반 스킬 레이팅 시스템 ⭐⭐⭐
**목적**: 공정한 매칭 및 사용자 동기부여
- 사용자별 ELO 점수 (초기값: 1500)
- 경기 결과 입력 시스템
- 자동 레이팅 계산
- 레이팅 히스토리 추적
- 등급 자동 조정 (E~S)

**구현 세부사항**:
```typescript
// ELO 계산 공식
function calculateElo(
  currentRating: number,
  opponentRating: number,
  result: 'win' | 'loss' | 'draw',
  kFactor = 32
): number
```

#### 1.3 포인트 & 리워드 시스템 ⭐⭐⭐
**목적**: 사용자 참여 증대 및 수익화 기반
- 포인트 획득:
  - 모임 참여: 50P
  - 리뷰 작성: 30P
  - 게시글 작성: 10P
  - 댓글 작성: 5P
  - 출석 체크: 5P/일
  - 친구 초대: 100P
  - 프로필 완성: 50P
- 포인트 사용:
  - 모임 참가비 할인
  - 프리미엄 기능 해금
  - 배지 구매
  - 기프트 교환
- 포인트 히스토리 및 만료 관리

#### 1.4 게이미피케이션 시스템 ⭐⭐⭐
**목적**: 재방문율 증가 및 참여 유도

**배지 시스템**:
- 🏅 신규 회원 (회원가입)
- 🏸 첫 경기 (첫 모임 참여)
- ⭐ 베테랑 (50회 참여)
- 👑 챔피언 (100회 참여)
- 📝 작가 (게시글 50개)
- 💬 소통왕 (댓글 100개)
- 🎯 정확한 승부사 (승률 70% 이상)
- 🔥 연속 출석 (7일, 30일, 100일)

**도전과제**:
- 이번 주 3회 참여하기
- 새로운 지역에서 경기하기
- 5명에게 좋은 평가 받기
- 다양한 레벨과 경기하기

**리더보드**:
- 전체 랭킹 (ELO 기준)
- 지역별 랭킹
- 연령대별 랭킹
- 월간 활동 랭킹
- 클럽별 랭킹

#### 1.5 지능형 매칭 시스템 ⭐⭐⭐
**목적**: 적합한 파트너/모임 자동 추천
- ELO 점수 기반 매칭 (±200 범위)
- 선호 지역 우선
- 활동 시간대 분석
- 성향 매칭 (공격형/수비형)
- 매칭 성공률 추적
